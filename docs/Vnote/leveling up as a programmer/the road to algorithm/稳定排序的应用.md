**题目**
将一个数组前半部分全部变成偶数,后半部分变成奇数

```java
    public static void BubbleSort(int[] arr){
        for(int i=0; i<arr.length;i++){
            for(int j=0; j<arr.length-1-i;j++){
                int temp=0;
                // 前偶后奇就交换
                if(arr[j] %2==0 && arr[j+1] %2==1){
                    temp = arr[j+1];
                    arr[j+1] = arr[j];
                    arr[j]= temp; 
                }
            }
        }
        for(int i=0;i<arr.length;i++){
            System.out.print(arr[i]+" ");            
        }
    }
```
- 直接插入排序
```java
    public static void InsertSort(int[] arr){
        int nextOddIndex=0;
        int possibleOddIndex=0;
        for(int i=0;i<arr.length;i++){
            if(arr[i]%2==1){
                nextOddIndex = i;
                while(nextOddIndex > possibleOddIndex){
                    int temp = arr[nextOddIndex];
                    arr[nextOddIndex] = arr[nextOddIndex-1];
                    arr[nextOddIndex-1] = temp;
                    nextOddIndex--;
                }
                possibleOddIndex++;
            }
        }
        for(int i=0;i<arr.length;i++){
            System.out.print(arr[i]+" ");            
        }
    } 
```

```java
    public static void SpaceSort(int[] arr){
        int oddBegin=0,oddEnd=0;
        for(int i=0;i<arr.length;i++){
            if(arr[i]%2==0)
                oddEnd++;
        }
        int[] sorted = new int[arr.length];
        for(int i=0;i<arr.length;i++){
            if(arr[i] %2== 1){
                sorted[oddBegin] =arr[i];
                oddBegin++;
            }else{
                sorted[oddEnd] = arr[i];
                oddEnd++;
            }
        }
        arr = sorted;
        for(int i=0;i<arr.length;i++){
            System.out.print(arr[i]+" ");            
        }
    }
}
```
