
首先，我们会给`map`数组一个更具描述性的名字，比如`lastSeen`，这表示字符上次出现的位置。然后，我们将主要的逻辑分割成几个子方法，每个子方法都有其专门的功能。
代码如下所示：
```java
import java.util.Arrays;
public class LongestSubstringFinder {
    /**
     * Finds the length of the longest substring without repeating characters.
     * 
     * @param s the input string
     * @return the length of the longest substring
     */
    public int lengthOfLongestSubstring(String s) {
        int[] lastSeen = new int[128];
        Arrays.fill(lastSeen, -1);
        int leftPointer = 0;
        int maxLen = 0;
        for (int rightPointer = 0; rightPointer < s.length(); rightPointer++) {
            leftPointer = updateLeftPointer(s, lastSeen, leftPointer, rightPointer);
            lastSeen[s.charAt(rightPointer)] = rightPointer;
            maxLen = updateMaxLength(leftPointer, rightPointer, maxLen);
        }
        return maxLen;
    }
    /**
     * Updates the position of the left pointer if the current character has been seen recently.
     *
     * @param s the input string
     * @param lastSeen the array that holds the last seen positions of the characters
     * @param leftPointer the current left pointer
     * @param rightPointer the current right pointer
     * @return the updated left pointer
     */
    private int updateLeftPointer(String s, int[] lastSeen, int leftPointer, int rightPointer) {
        return Math.max(lastSeen[s.charAt(rightPointer)] + 1, leftPointer);
    }
    /**
     * Updates the maximum length of the substring seen so far.
     *
     * @param leftPointer the current left pointer
     * @param rightPointer the current right pointer
     * @param maxLen the current maximum length
     * @return the updated maximum length
     */
    private int updateMaxLength(int leftPointer, int rightPointer, int maxLen) {
        return Math.max(rightPointer - leftPointer + 1, maxLen);
    }
}
```
`updateLeftPointer`负责更新左指针的位置，`updateMaxLength`负责更新无重复子串的最长长度。

以上是针对滑动窗口算法处理字符串中无重复字符最长子串问题的重构案例。对应的时间复杂度是O(n)，其中n是输入字符串的长度，因为每个字符只被访问一次。空间复杂度是O(1)，因为`lastSeen`数组的大小是固定的，与字符集的大小相关，与输入字符串的长度无关。
